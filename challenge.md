# Coding Challenge: Counter Game

## Obiettivo
Creare un gioco semplice che metta alla prova la tua conoscenza di HTML, CSS e JavaScript fondamentale.

## Requisiti
1. Crea una pagina con:
   - Un display che mostra un numero (partendo da 0)
   - Un pulsante "+" che incrementa il numero
   - Un pulsante "-" che decrementa il numero
   - Un pulsante "Reset" che riporta il numero a 0

2. Regole del gioco:
   - Il contatore non può andare sotto lo 0
   - Ogni 10 click sul pulsante "+", il numero deve diventare rosso
   - Quando il numero è 0, il pulsante "-" deve essere disabilitato
   - Il pulsante Reset deve essere attivo solo quando il numero è diverso da 0

3. Styling:
   - I pulsanti devono avere un effetto hover
   - Il display del numero deve essere centrato e grande
   - Il layout deve essere responsive

## Bonus (Sfide Extra)
- Aggiungere un effetto di transizione quando il numero cambia
- Implementare un contatore di click totali
- Aggiungere un sistema di punteggio basato sulla velocità dei click

## Suggerimenti
- Usa addEventListener per gestire i click
- Ricordati di manipolare il DOM per aggiornare il display
- Utilizza le classi CSS per gestire gli stili dinamici

## Conoscenze Richieste
- HTML: struttura base, elementi button
- CSS: selettori, proprietà base, hover effects
- JavaScript: 
  - Variabili
  - Eventi DOM
  - Operatori
  - Condizionali
  - Manipolazione del DOM
